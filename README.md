# lumen-be
A mysql and php backend based on Lumen framework

## Instalation

**MYSQL SERVER INSTALATION**  
brew install MYSQL  

**To start and stop the service**    
mysql.server start   
mysql.server stop


**Install dependencies:**  
composer install  

php artisan jwt:secret  
  
Run the PHP server:  
php -S localhost:8000 -t public
